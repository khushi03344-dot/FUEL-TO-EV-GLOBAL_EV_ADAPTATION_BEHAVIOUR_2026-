Project Description

The Agriculture Crop Yield Analysis Dashboard is an interactive Power BI project developed to analyze agricultural production and crop yield using historical crop data. The project transforms raw agricultural data into meaningful insights through data cleaning, data modeling, DAX calculations, and interactive dashboard visualizations.

The dashboard helps users understand crop performance, production trends, rainfall impact, fertilizer usage, seasonal variations, and state-wise agricultural productivity.

Project Objectives

Analyze crop yield across different crops and states.
Identify the highest and lowest yielding crops.
Compare production among different Indian states.
Study the impact of rainfall on crop yield.
Analyze fertilizer usage and production.
Compare production across seasons.
Build an interactive Power BI dashboard for agricultural analysis.
